# Customer Data Analysis

##  Project Overview
This project analyzes customer and salary data to discover patterns and business insights using Excel.

##  Business Problem
Companies need to understand factors affecting salary levels to improve hiring decisions and cost management.

##  Key Insights
- Salary increases with experience level
- Higher education levels tend to have higher salaries
- Some salary ranges are highly concentrated

##  Business Recommendations
- Focus hiring on mid-high experience levels
- Review salary structure for low experience employees
- Improve hiring strategy based on education level
